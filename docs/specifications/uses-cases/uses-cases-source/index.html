Étude de Cas : afficher_source_depliee

Action utilisateur


Précondition :

source définie dans le profil utilisateur & dépliée


Flux Normal :

Si le ttl (source controlée) est dépassé ou nouvelle session utilisateur (source personelle) alors appel de "rafraichir_source" et de "recherche_xslt_xpath" (en lui fournissant la DTD ou l'URL de la xslt) ; Puis, affichage du flux XML avec la feuille XSLT, en tenant compte du marquage sur les items lus.


Étude de Cas : afficher_source_pliee

Action utilisateur


Précondition :

source définie dans le profil utilisateur & pliée


Flux Normal :

Si ttl est dépassé (source controllée) ou nouvelle session utilisateur (source personnelle) alors appel de "rafraichir_source" et de "recherche_xslt_xpath" (en lui fournissant la DTD ou l'URL de la xslt) ; Puis affichage du titre de la source et un marquage si elle contient des items non lus.


Étude de Cas : rafraichir_source

Action utilisateur


Précondition :

source définie dans le profil utilisateur & visible


Flux Normal :

Si la source est controllée d'accessibilité CAS alors appel de "recuperer_pt_cas" ; Puis, si la source est individualisée alors appel de "charger_source_individuelle" sinon "charger_source_globale" en leur donnant le proxy ticket si besoin.


Flux Alternatif1 :

Erreur si source inaccessible et invisible pour l'utilisateur.

Étude de Cas : recherche_xslt_xpath

Action globale


Précondition :

DTD ou (URL de la feuille XSLT et xpath) connus & fichier de mapping défini si DTD


Flux Normal :

Si on a l'URL alors chargement de la feuille XSLT ; Si on n'a pas l'xpath alors on le récupère via la DTD ; Si on a la DTD alors lecture du fichier de mapping ; Puis recherche de la feuille XSLT et de l'xpath.

Étude de Cas : enregistrer_source

Action utilisateur

Étude de Cas : charger_source_globale

Action globale : charger une source controllée non individualisée.


Précondition :

Flux Normal :

Accède au serveur distant en accès public ou CAS ; Si c'est CAS, il lui fournit le proxy ticket ; Chargement en mémoire du flux XML de la source controllée non individualisée ; Remet le timer global de la source à 0.
